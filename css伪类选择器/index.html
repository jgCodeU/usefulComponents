非特定类型元素

/* 非特定类元素 */
div:not(.special) {
    border: 1px solid #e5e7eb;
    transition: all 0.3s ease;
}
div:not(.special):hover {
    border-color: #3b82f6;
    box-shadow: 0 0 0 2px rgba(59, 130, 246, 0.2);
}

/* 非特定类型元素 */
input:not([type="submit"]):not([type="reset"]) {
    background-color: #f9fafb;
    border: 1px solid #d1d5db;
}
input:not([type="submit"]):not([type="reset"]):focus {
    outline: none;
    border-color: #3b82f6;
    box-shadow: 0 0 0 2px rgba(59, 130, 246, 0.2);
}

/* 表单元素状态 */

}

    background-color: #f3f4f6;

}

/* 输入验证状态 */

    border-color: #ef4444;
}

}

/* 选中状态 */
input[type="radio"]:checked + span,

    font-weight: 600;
    color: #3b82f6;
}

标签 2 内容

这是标签 2 的内容。使用:target伪类可以轻松实现标签切换效果，而无需编写JavaScript代码。

标签 3 内容

这是标签 3 的内容。这种实现方式非常适合简单的标签切换功能，代码简洁且易于维护。

/* 目标伪类实现标签切换 */
.tab-pane {
    display: none;
}
.tab-pane:target {
    display: block;
    animation: fadeIn 0.5s ease-in-out;
}
@keyframes fadeIn {
    from { opacity: 0; transform: translateY(10px); }
    to { opacity: 1; transform: translateY(0); }
}

/* 激活的标签链接样式 */
.tab-link[href=""] {
    background-color: #e0e7ff;
    color: #1e40af;
    font-weight: 600;
}
.tab-link[href="#tab1"]:target,
.tab-link[href="#tab2"]:target,
.tab-link[href="#tab3"]:target {
    background-color: #e0e7ff;
    color: #1e40af;
    font-weight: 600;
}
